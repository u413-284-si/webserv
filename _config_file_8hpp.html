std::ostream& operator<< ( std::ostream &  ostream,
Method  method 
)

Overload << operator to a Method.

Translates the Method enum to a string. Since Method is an enum it should only contain valid values. On the off chance of memory bugs, it's asserted that the value is in range of the enum. If the Method is MethodCount, it will output "enum MethodCount".
