Detailed Description

Logger class to log messages.

Implements the Singleton pattern to log messages to different outputters. The Logger class is a Singleton class, so it can be accessed from anywhere in the code. The Logger class can log messages to different outputters, which implement the ILogOutputter interface. The Logger class can log messages with different log levels, which are defined in the LogLevel enum. The Logger class can be initialized with a log level and an outputter.

void weblog::Logger::operator+= ( const LogData &  logData) const

Log a message with the given log data.

This overload passes the LogData object to all registered the log outputters. The log message is only logged if the log level of the log data is greater or equal to the log level of the Logger object.
