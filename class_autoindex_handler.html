std::string AutoindexHandler::execute ( const std::string &  path,
const std::string &  uriPath 
)

Execute the autoindex handler.

Generates an HTML response with the contents of a directory. The response contains a table with the file names, last modified time and file size. The file names are relative links to the files constructed with the uriPath parameter. If a function throws, returns empty string.

Parameters
pathPath to directory.
uriPathURI path which lead to the directory.
Returns
std::string HTML response.
