std::string AutoindexHandler::execute ( const std::string &  path)

Execute the autoindex handler.

Generates an HTML response with the contents of a directory. The response contains a table with the file names, last modified time and file size. The file names are links to the files. If a function throws, returns empty string.

Parameters
pathPath to directory.
Returns
std::string HTML response.
