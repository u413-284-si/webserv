Selects the server configuration for a connection.

Finds possible server configurations by calling findMatchingServerConfigs(). If no matching server configuration is found, returns false. If more than one matching server is found, sets the first one found. Also sets the first location of the found server configuration.

bool hasValidServerConfig ( Connection &  connection,
const std::vector< ConfigServer > &  serverConfigs,
const std::string &  host 
)

Overload for hasValidServerConfig() to aditionally check a server configuration for host.

Works similar to hasValidServerConfig(). If after finding all matching server configurations more than one match was found, it iterates through the matches and checks if the server name matches the host. If it does, sets the found server config. Also sets the first location of the found server configuration. If no server name matches the host, the first found match is kept as matching server.

Parameters
connectionThe connection object to select the server configuration for.
serverConfigsThe vector of server configurations.
hostThe host name to match with the server name.
Returns
true if a valid server configuration is found, false otherwise.
