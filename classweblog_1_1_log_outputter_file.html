void weblog::LogOutputterFile::log ( const LogData &  logData)
virtual

Log a message to the file.

The log function logs the message to the file. If the bool m_isOpen is false, the file is opened. If the file could not be opened, an error message is printed to std::cerr. The message is formatted with the getFormattedMessage function. After every log message, the file is flushed.

Parameters
logDataThe log data to log.
