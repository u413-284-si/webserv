Detailed Description

Class for C functions on the file system.

This class is a wrapper for C functions on the file system. It is used to make the code more testable.

FileSystemPolicy::fileType FileSystemPolicy::checkFileType ( const std::string &  path) const
virtual

Check the file type of a given path.

Uses stat() to check the file type of a given path. If the file does not exist, FileNotExist is returned. If the file is a regular file, FileRegular is returned. If the file is a directory, FileDirectory is returned. If the file is neither a regular file nor a directory, FileOther is returned.

Creates a file with the specified content at the given path.

This function attempts to create a file at the specified path and writes the provided content to it. If the file cannot be created, an error is logged and the function returns false. If the file exists already, the content is appended to the file.
