Server::Server ( )

Constructor for the Server class.

This constructor initializes a server instance by creating a server socket, using TCP protocol SOCK_STREAM, and setting it to non-blocking mode. It also creates an epoll instance for event handling. If any of these operations fail, the constructor throws a std::runtime_error. Then it bind() the server socket to a specific address and port, listen() on the server socket Then the server socket is added to the epoll instance.

Exceptions
std::runtime_errorif any of the socket creation, binding, listening, epoll creation, or epoll control operations fail.

Server::~Server ( )

Destructor for the Server class.

This destructor cleans up resources associated with the server instance by closing the server socket and the epoll instance.

The destructor performs the following cleanup actions:

  1. Closes the server socket to release the bound port and stop accepting new connections.
  2. Closes the epoll instance to release associated resources and stop monitoring events.

void Server::run ( )

Run the server event loop.

This method enters a continuous loop to handle incoming events using epoll. It waits for events using epoll_wait, processes the events, and dispatches them to appropriate handler functions.

Exceptions
std::runtime_errorif epoll_wait encounters an error.
