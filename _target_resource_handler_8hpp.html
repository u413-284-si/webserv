Iterate through Location objects to find longest matching path based on a given input path.

The longest matching path is the one that has the most characters in common with the input path. If no match is found, the function returns an iterator pointing to the end of the vector.

Parameters
locationsVector of Location objects.
pathString representing the path to be matched.
Returns
Constant_iterator to the location that has the longest matching path with input.
