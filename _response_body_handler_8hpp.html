std::string constructAllowHeader ( const bool(&)  allowedMethods[MethodCount])

Construct Allow header.

Constructs the Allow header based on the allowed methods. Methods are appended with ", " at the end to easily join them. If at the end at least one method was appended, the last ", " is removed. If no methods were appended, an empty string is returned.

Parameters

Returns
